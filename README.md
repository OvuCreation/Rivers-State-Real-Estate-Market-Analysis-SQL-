# Rivers State Real Estate Market Analysis

## Project Overview

This project analyzes the real estate market in Rivers State, Nigeria, with a focus on selected neighborhoods in Port Harcourt and Obio-Akpor. The objective is to uncover pricing patterns, market trends, and potential investment opportunities using data analytics tools.

## Business Problem

Real estate stakeholders often struggle to determine whether properties are fairly priced and which areas offer the best investment opportunities. This project uses historical property transactions to provide data-driven insights that support decision-making.

## Dataset

The analysis was conducted using a simulated Rivers State real estate dataset containing:

* 12 neighborhoods across Port Harcourt and Obio-Akpor
* 259 property listings
* 729 transaction records spanning multiple years

The dataset includes information such as:

* Property type
* Property size (square meters)
* Listing price
* Transaction price
* Transaction date
* Neighborhood location

## Tools Used

* **Python** (Pandas, Matplotlib, Seaborn)

* **Power BI**
* **VS Code**
* **GitHub**

## Project Workflow

1. Generated and prepared the dataset.
2. Performed data cleaning and exploratory data analysis using Python.
3. Designed a relational database and conducted business analysis using SQL.
4. Created visualizations to communicate key findings.

## Key Questions Answered

* Which neighborhoods have the highest price per square meter?
* Which property categories generate the highest revenue?
* How has the real estate market changed over time?
